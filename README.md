# Genisys-inst
Perfect Genisys Installer!

How to

1,Download zip from Github.

2,Unzip it.

3,Run Installer.exe

4,Select language and PHP version

5,Select Pocketmine-MP forks

6,You only wait a minute,and your Genisys start.

Enjoy your Minecraft PE!
